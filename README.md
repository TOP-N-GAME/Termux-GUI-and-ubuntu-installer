# Termux-GUI-and-ubuntu-installer

Для начала вам нужно установить стабильную версию Termux с github: https://github.com/termux/termux-app/releases

А так-же Termux:X11: https://github.com/termux/termux-x11/releases

Затем вы должны установить install.sh с этого репозитория и оставить его в вашей папке загрузок.

Теперь в Termux вы должны перекинуть этот файл в домашнюю директорию, сначала разрешите доступ к хранилищу:
```
termux-setup-storage
```

Теперь файл можно найти в появившейся папке storage, через неё можно получить доступ к некоторым папкам на вашем устройстве, найдите файл install.sh и копируйте его в домашнюю директорию
