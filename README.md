# Termux-GUI-and-ubuntu-installer

Для начала вам нужно установить стабильную версию Termux с github: https://github.com/termux/termux-app/releases

А так-же Termux:X11: https://github.com/termux/termux-x11/releases

Затем вы должны установить install.sh с этого репозитория и оставить его в вашей папке загрузок.

Теперь в Termux вы должны скопировать этот файл в домашнюю директорию, сначала разрешите доступ к хранилищу:
```bash
termux-setup-storage
```

Теперь файл можно найти в появившейся папке storage, через неё можно получить доступ к некоторым папкам на вашем устройстве, найдите файл install.sh и копируйте его в домашнюю директорию.

Если файл лежит в папке загрузок то команда такая:
```bash
cp storage/downloads/install.sh ~/
```

после этого сделайте его исполняемым:
```bash
chmod +x install.sh
```
и запустите его:
```bash
./install.sh
```

После этого начнётся загрузка всего необходимого, потребуется скачать примерно 1 ГБ, и займёт на диске 3 ГБ

Termux может задавать вопросы при скачивании, отвечайте `y`

После завершения вы можете использовать следующие команды:<br>
`startubu` - для запуска ubuntu<br>
`startgui` - для запуска графического окружения.

Они создаются install.sh.

Графическое окружение универсально, и его можно запустить как из Termux так и из ubuntu. <br> В графическом окружении ubuntu не работают пакетные менеджеры а также система ошибочно думает что она Termux, в консольном режиме всё работает нормально, проблема только в графике с apt. <br> Во время работы графического окружения терминал Termux блокируется, чтобы увидеть графическое окружение перейдите в Termux:X11, чтобы восстановить работу терминала Termux отключите графическое окружение: <br> В Termux:X11 нажмите на кнопку справа сверху с именем текущего пользователя, нажмите на log out, в появившемся окне выберите log out, вернитесь в Termux

Для более лёгкой настройки, чтобы не было проблем с совместимостью и программы внутри ubuntu запускались без проблем, внутри ubuntu введите installenv, это вспомогательный скрипт, оно установит большинство всех необходимых библиотек для работы приложений
